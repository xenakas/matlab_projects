講義の進め方、成績評価：
  • 講義は、日本語で行う。英語の講義ノート（スライド）は、このウェブサイトに授業の前日（火曜日）の深夜までにアップロードする。講義ノートを印刷する場合は、１枚の紙に２ないし４スライドを印刷したり、両面印刷することで、紙やトナーを節約すること。Lecture notes are in English.
  • 評価は、宿題（50%）＋期末課題（50%）で行う。
  • 宿題は、２〜３週に一度出題する。
  • 期末課題は、DSGEモデルの分析を含んだマクロ経済学の学術論文を一本選び、その結果をコンピュータを使って再現（リプリケート）してもらう。なお、期末試験は行わない。
教科書：
  • McCandless, George (2008) The ABCs of RBCs: An Introduction to Dynamic Macroeconomic Models, Harvard University Press, Cambridge.
  • Gali, Jordi (2015) Monetary Policy, Inflation and the Business Cycle, second edition, Princeton University Press, Princeton.

  • Slides

  • Matlab code for stochastic simulation solow.m

Reading: McCandless Ch.3
  • Slides (incl. Assignment #2)
  • Intertemporal decision on consumption and saving (for review) slides

Part II. Real Business Cycles (RBC)

5. 11/4 Dynamic Programming

Reading: Cooper (2013, lecture note), McCandless Ch.4 and McCandless Ch.5
  • Slides (incl. Assignment #3)
  • Matlab codes
    • Successive approximation: dp_simple.m
    • Linear interpolation for deterministic model: dp.m / valfun.m / golden.m
    • For stochastic model: dpsto.m / valfunsto.m
6. 11/18 Stochastic neoclassical growth model, Hansen's RBC model Assignment #3 Due

Reading: McCandless Ch.6 and Hansen (1985)
  • Slides (incl. Assignment #4)
  • A primer for difference equations and time series slides
  • Excel file for the stochastic growth model
  • Matlab codes
    • Blanchad-Khan's method for the stochastic growth model bk_simple.m
    • Uhlig's method for the stochastic growth model uhlig_simple.m
    • Dynare code for the stochastic growth model rbc_simple.mod / hpfilter.m
7. 12/2 Linear Quadratic (LQ) Dynamic Programming

Reading: McCandless Ch.7
  • Slides
  • Matlab codes
    • Solving the Ricatti equation of the neoclassical growth models 
      • Linear: lq_simple.m / lq_simple_sym.m (requires Symbolic Math Toolbox)
      • Log-linear: lq_simple_log.m / lq_simple_log_sym.m
      • Log-linear with stoch. TFP: lq_simple_logstoch.m / lq_simple_logstoch_sym.m
    • Dynare code for the stochastic growth model (using nonlinear equations) rbc_simple_nl.mod / hpfilter.m
